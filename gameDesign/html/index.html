Common tags and elements:

  • h1 - h8 are heading tags. The higher the number, the smaller the heading.
  • A div is a plain element. Use it to create arbitrary blocks to attach styles onto.
  • A img tag displays an image; it's "src" attribute specifies the name and location of the image.
  • The ul and li tags create bulleted lists. li tags are sandwiched by ul tags.
  • The b tag lets you bold bits of text.
  • The i tag displays text in italics.

Common CSS attributes:

  • font-family lets you change an element's font.
  • font-size lets you change an element's text size.
  • color lets you change the text's color.
  • background-color changes an element's background color.
  • margin lets you specify the element's margin in pixels, e.g. 100px.
  • border lets you give an element a border, e.g. 4px solid black.
  • padding lets you specify space between the element's border and its content, e.g. 10px.
  • float lets you place divs side by side. You can set it to "left" or "right".
